# Bookswap app

#### This is a project idea for the final module of my SDAcademy course.


### Project goals:
- List books added by users
- Help user swap books between them

### Main functionalities:

- Main page:
    - List of books 
    - Search bar 
    - Registration
    - Add book if user is logged in
    
- Admin panel:
    - superusers login
    - add categories for books
    - deletes users
